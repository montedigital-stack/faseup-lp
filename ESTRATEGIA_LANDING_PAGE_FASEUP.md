## 📱 **Fluxo Até Check-out**

### **Jornada do Usuário:**
```
1. Landing Page → Interesse desperto
2. CTA Click → Decisão de apoiar/testar
3. Formulário → Dados básicos (nome, email, empresa)
4. Seleção de Plano → Apoiador vs Gratuito
5. Pagamento → InfinityPay checkout (CPF)
6. Confirmação → Email + acesso app
7. Onboarding → Tutorial personalizado
8. Grupo VIP → WhatsApp dos apoiadores
```
