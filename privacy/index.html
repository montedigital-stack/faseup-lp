🔒 Política de Privacidade - FaseUp

Última atualização: 4 de setembro de 2025

📱 Introdução

O FaseUp ("nós", "nosso" ou "aplicativo") está comprometido em proteger sua privacidade. Esta política descreve como coletamos, usamos e protegemos suas informações.

📊 Informações que Coletamos

Dados de Conta

  • Nome e endereço de email
  • Informações de perfil profissional
  • Função na obra (gestor, operário, cliente)

Dados dos Projetos

  • Informações das obras
  • Fotos e documentos
  • Cronogramas e orçamentos
  • Comunicações do projeto

Dados de Localização

  • Geolocalização das fotos (com sua permissão)
  • Localização das obras

Dados de Uso

  • Como você usa o aplicativo
  • Recursos mais utilizados
  • Dados de performance

🎯 Como Usamos seus Dados

  • Funcionalidade: Prover os serviços do aplicativo
  • Comunicação: Notificações sobre projetos
  • Melhoria: Aprimorar recursos usando IA
  • Segurança: Proteger sua conta e dados

🤖 Inteligência Artificial

Usamos IA para:

  • Análise de progresso de obras
  • Otimização de recursos
  • Previsões de cronograma
  • Relatórios automatizados
Seus dados são processados localmente sempre que possível.

📱 Permissões do App

  • Câmera: Para fotografar o progresso das obras
  • Galeria de Fotos: Para selecionar fotos existentes
  • Localização: Para geotagging de fotos (opcional)
  • Armazenamento: Para salvar dados offline

🔐 Compartilhamento de Dados

NÃO vendemos seus dados.

Compartilhamos apenas:

  • Com membros autorizados do seu projeto
  • Com provedores de serviço (hospedagem, análise)
  • Quando exigido por lei

🛡️ Segurança

  • Dados criptografados
  • Conexões HTTPS seguras
  • Backup regular e seguro
  • Controle de acesso rigoroso

📍 Dados de Localização

  • Usado apenas para geotagging de fotos
  • Nunca rastreamos sua localização
  • Você pode desativar a qualquer momento

👶 Menores de Idade

Este aplicativo não é destinado a menores de 13 anos.

🇧🇷 Conformidade com LGPD

Você tem direito a:

  • Acessar seus dados
  • Corrigir informações
  • Excluir sua conta
  • Portabilidade de dados

📞 Contato

Para questões sobre privacidade:

  • Email: privacy@faseup.com
  • Website: www.faseup.com

🔄 Alterações

Esta política pode ser atualizada. Notificaremos sobre mudanças significativas.

Ao usar o FaseUp, você concorda com esta política de privacidade.
